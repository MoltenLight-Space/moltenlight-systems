# Business Case

!!! note "Status"
    **Skeleton, v0.1.** Phase 0 artifact. To be populated before MDR.

This document makes the case for *doing the project at all*. Molten Light is
not a commercial venture in the traditional sense; the business case is
therefore framed in **non-financial ROI** terms, with direct costs bounded
but not primary. The question is: is the total return — engineering,
educational, community, personal — worth the committed effort?
